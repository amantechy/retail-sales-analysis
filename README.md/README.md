# Retail Sales Analysis Project

## Objective
This project analyzes the Superstore retail dataset to identify sales trends, profitable categories, and key customer insights using Python and Pandas.

---

## Dataset Information
- Dataset: Sample Superstore Sales Data
- Rows: 51,290
- Columns: 21

---

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib

---

## Key Business Insights
1. Technology and Office Supplies have higher profit margins (~14%) compared to Furniture (~7%).
2. Copiers and Phones are the most profit-driving sub-categories.

---

## Analysis Performed
- Data Cleaning (date conversion, numeric conversion)
- Monthly Sales Trend Analysis
- Top Products and Customers Analysis
- Category-wise and Region-wise Performance
- Profit Margin Calculation
